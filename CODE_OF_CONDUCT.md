# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

- Be respectful and constructive.
- Assume good intent and discuss ideas on technical merit.
- Avoid harassment, personal attacks, and discriminatory language.

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and other community channels.

## Reporting

To report unacceptable behavior, contact: `info@gitopshq.io`.

All reports will be reviewed and handled in good faith.
